# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue.**

Report it privately to: **security@royalda.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect an acknowledgement within **48 hours** and a resolution update within **7 days**.

## Security Practices

- All secrets are managed via environment variables (never committed to source)
- Dependencies are monitored with GitHub Dependabot
- Secret scanning and push protection is enabled on this repository
